# IPA Input Shortcuts for French Trainer

Type letter + number to convert to IPA characters in real-time.

## Vowels

### A variants
- `a1` → ɑ (a script)
- `a2` → æ (ae ligature)
- `a3` → ɐ (a upside down)
- `a4` → ɑ̃ (a script nasalized)

### E variants
- `e1` → ə (schwa)
- `e2` → ɛ (epsilon)
- `e3` → ɜ (epsilon reversed)
- `e4` → ɝ (epsilon reversed rhotic)
- `e5` → ɘ (schwa closed)
- `e6` → ɞ (epsilon reversed closed)
- `e7` → ɛ̃ (epsilon nasalized)
- `e8` → ɚ (schwa rhotic)

### I variants
- `i1` → ɪ (small capital I)
- `i2` → ɨ (i barred)
- `i3` → ɪ̈ (small capital I diaeresis)

### O variants
- `o1` → ɔ (open o)
- `o2` → ɔ̃ (open o nasalized)
- `o3` → ø (o slash)
- `o4` → œ (oe ligature)
- `o5` → ɶ (small capital OE)

### U variants
- `u1` → ʊ (upsilon)
- `u2` → ʉ (u barred)
- `u3` → ɥ (u with hook)

### Y variants
- `y1` → ʏ (small capital Y)
- `y2` → ʎ (y upside down)
- `y3` → ɣ (gamma)
- `y4` → ɤ (ram's horns)

## Consonants

### B variants
- `b1` → β (beta)
- `b2` → ɓ (b hook)
- `b3` → ʙ (small capital B)

### C variants
- `c1` → ç (c cedilla)
- `c2` → ɕ (c curl)

### D variants
- `d1` → ð (eth)
- `d2` → ɗ (d hook)
- `d3` → ɖ (d retroflex hook)

### G variants
- `g1` → ɡ (g script)
- `g2` → ɠ (g hook)
- `g3` → ɢ (small capital G)
- `g4` → ʛ (small capital G hook)

### H variants
- `h1` → ħ (h stroke)
- `h2` → ɦ (h hook)
- `h3` → ɥ (h turned)
- `h4` → ɧ (h hook stroke)
- `h5` → ʜ (small capital H)

### J variants
- `j1` → ɟ (j barred)
- `j2` → ʄ (j hook)

### L variants
- `l1` → ɫ (l tilde)
- `l2` → ɭ (l retroflex hook)
- `l3` → ɬ (l belt)
- `l4` → ʟ (small capital L)
- `l5` → ɮ (lezh)

### M variants
- `m1` → ɱ (m hook)

### N variants
- `n1` → ŋ (eng)
- `n2` → ɲ (n left hook)
- `n3` → ɳ (n retroflex hook)
- `n4` → ɴ (small capital N)

### P variants
- `p1` → ɸ (phi)

### R variants
- `r1` → ɾ (r tap)
- `r2` → ɹ (r turned)
- `r3` → ʀ (small capital R)
- `r4` → ʁ (small capital inverted R)
- `r5` → ɼ (r long leg)
- `r6` → ɽ (r retroflex hook)
- `r7` → ɺ (r turned long leg)

### S variants
- `s1` → ʃ (esh / sh)
- `s2` → ʂ (s retroflex hook)

### T variants
- `t1` → θ (theta)
- `t2` → ʈ (t retroflex hook)

### V variants
- `v1` → ʌ (turned v / wedge)
- `v2` → ʋ (v hook)
- `v3` → ⱱ (v small hook)

### W variants
- `w1` → ʍ (w turned)

### X variants
- `x1` → χ (chi)

### Z variants
- `z1` → ʒ (ezh / zh)
- `z2` → ʐ (z retroflex hook)
- `z3` → ʑ (z curl)

## Special Symbols

### Stress markers
- `q1` → ˈ (primary stress)
- `q2` → ˌ (secondary stress)

### Length markers
- `k1` → ː (length mark / triangular colon)
- `k2` → ˑ (half-length mark)

## Examples

**Type:** `s1jɛ̃` → **Result:** ʃjɛ̃ (chien - dog)

**Type:** `kwa.sa1n1` → **Result:** kwa.sɑ̃ŋ (croissant)

**Type:** `bo1n1.z1uq1k1` → **Result:** boɑŋ.ʒuˈː (bonjour)

## Tips

- Type naturally and the conversion happens in real-time
- No need to type IPA slashes `/` - they're stripped automatically when checking
- Most common French IPA characters:
  - `e1` (ə - schwa)
  - `e2` (ɛ - epsilon)
  - `o1` (ɔ - open o)
  - `s1` (ʃ - sh sound)
  - `z1` (ʒ - zh sound)
  - `n1` (ŋ - ng sound)
  - `a1` (ɑ - open a)
  - `k1` (ː - length mark)
